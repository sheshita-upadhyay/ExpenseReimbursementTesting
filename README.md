# Expense Reimbursement Software Testing Project

This project is a **Software Testing Project** for an **Expense Reimbursement Application**.  
It demonstrates **manual testing**, **API testing**, **database testing**, and **bug tracking** using modern tools and practices.

---

## Features
- Manual testing with **Excel test cases**, **bug reports**, and **test summary**
- API testing using **Postman**:
  - GET, POST, PATCH, DELETE operations
  - **Postman Collection**
  - **Mock Server** for testing APIs without backend
  - **Environment setup** for dynamic variables
- **PostgreSQL** database for storing reimbursement data
- **JIRA** used for bug tracking and project management

---

## Technologies & Tools Used
- **Excel** – Manual test cases, bug report, test summary
- **Postman** – API testing, collections, mock server, environment variables
- **PostgreSQL** – Database testing
- **JIRA** – Bug tracking
- **JavaScript** – For test scripts and automation logic (if any)

## How to Use
1. Open **Excel files** to view manual test cases, bug reports, and test summary.
2. Open **Postman** to:
   - Import the collection
   - Use the environment for API testing
   - Test APIs using the mock server or actual backend
3. Use **PostgreSQL** to run database scripts and validate data.
4. Refer to **JIRA** for bug tracking and project management workflow.
